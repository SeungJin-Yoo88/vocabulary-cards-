# 📊 GitHub Pages vs Vercel 비교

## 🟢 GitHub Pages에서 작동하는 기능

### ✅ 완벽하게 작동 (서버 불필요)
1. **🔊 음성 발음** - Web Speech API (브라우저 기능)
2. **🏷️ 카테고리 관리** - localStorage 저장
3. **🧠 SRS 복습 시스템** - 클라이언트 사이드 계산
4. **📊 학습 통계** - 로컬 데이터 추적
5. **💾 Import/Export** - 파일 다운로드/업로드
6. **🌙 다크모드** - CSS 테마 전환
7. **📱 PWA 기본 기능** - 오프라인 캐싱 (HTTPS 필요)
8. **⭐ 즐겨찾기** - 로컬 저장
9. **🔍 검색** - 클라이언트 사이드 필터링
10. **✏️ 수동 카드 추가** - 모든 필드 직접 입력

### ⚠️ 제한사항
- **❌ AI 자동 채우기** - Serverless function 필요 (GitHub Pages 미지원)
- **⚠️ PWA 설치** - GitHub Pages는 HTTPS이지만 일부 제한

## 🟡 Vercel 배포 시 추가 기능

### ✨ 추가로 작동하는 기능
1. **🤖 AI 자동 채우기** - Claude API로 자동 생성
   - 단어만 입력 → 의미/예문/발음/팁 자동 생성
   - 한글 입력 → 영어 번역 + 정보 생성
   
2. **📱 완전한 PWA** - 앱 설치 프롬프트
   - "📱 앱으로 설치" 버튼
   - 독립 실행 창
   - 더 나은 오프라인 지원

## 🎯 권장 사항

### 옵션 1: GitHub Pages만 사용 (현재)
**장점:**
- ✅ 무료
- ✅ 설정 필요 없음
- ✅ 이미 배포됨
- ✅ 대부분의 기능 작동

**단점:**
- ❌ AI 자동 채우기 불가
- ⚠️ 수동으로 카드 정보 입력 필요

**추천 대상:**
- API 키 없이 바로 사용하고 싶은 경우
- 수동 입력이 괜찮은 경우
- 무료로만 사용하고 싶은 경우

---

### 옵션 2: Vercel 추가 배포 (권장)
**장점:**
- ✅ GitHub Pages의 모든 기능
- ✅ AI 자동 채우기 추가
- ✅ 카드 추가 시간 90% 단축
- ✅ 더 나은 PWA 지원
- ✅ 무료 (Hobby 플랜)

**단점:**
- ⚠️ Anthropic API 키 필요 (무료 $5 크레딧)
- ⚠️ 5분 설정 시간 필요

**추천 대상:**
- AI 자동 채우기를 사용하고 싶은 경우
- 빠르게 많은 카드를 추가하고 싶은 경우
- 최고의 사용자 경험을 원하는 경우

**작동 방식:**

**소요 시간:** 카드당 10-30초

## 🔄 두 가지 다 사용하기 (Best Practice)

### 전략: 병행 사용
1. **GitHub Pages**: 일상적인 학습 및 복습
   - 빠른 접근
   - 데이터는 로컬 저장 (브라우저)
   
2. **Vercel**: 새 카드 대량 추가
   - AI로 빠르게 생성
   - Export로 백업
   - GitHub Pages로 Import

### 데이터 동기화 방법
```
Vercel에서 카드 생성
    ↓
⚙️ 설정 → 📤 카드 내보내기
    ↓
vocabulary-backup.json 다운로드
    ↓
GitHub Pages에서
    ↓
⚙️ 설정 → 📥 카드 가져오기
    ↓
JSON 파일 업로드
```

## 📊 비용 비교

### GitHub Pages
- **비용**: 완전 무료
- **제한**: 100GB/월 대역폭
- **기능**: 정적 호스팅만

### Vercel (Hobby 플랜)
- **비용**: 완전 무료
- **제한**: 
  - 100GB/월 대역폭
  - 100개/일 Serverless 실행
- **기능**: Serverless Functions 지원

### Anthropic API
- **무료 크레딧**: $5
- **카드 생성**: 약 1,000~1,500개
- **이후 비용**: 카드당 약 $0.003~0.01
